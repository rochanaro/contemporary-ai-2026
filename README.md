# Data Science and AI Bootcamp - 2026
## Module 5 - Fundamentals of Contemporary AI (Practical)

This practical session compares three different approaches to solving the same image classification task using the MNIST handwritten digits dataset:

- **Classical Machine Learning:** Logistic Regression
- **Deep Learning:** Multi-Layer Perceptron (PyTorch)
- **Zero-Shot AI:** Vision-Language Model (choose either **CLIP** or **BLIP-VQA**)

Using the same dataset throughout allows you to compare the strengths, limitations, and performance of each approach under identical conditions.

As a bonus, the notebook also demonstrates speech generation using an open-source text-to-speech model.

## Running the Notebook

This notebook is intended to run on **Google Colab** with a GPU runtime.

## Learning Outcomes

By the end of this practical, you will be able to:

- Train and evaluate a classical machine learning classifier.
- Build and train a simple neural network with PyTorch.
- Perform zero-shot image classification using a Vision-Language Model.
- Compare different AI paradigms for the same problem.
- Generate speech using an open-source text-to-speech model.
